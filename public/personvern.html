Vi er behandlingsansvarlig for personopplysningene som behandles på Skatteberegner.no i henhold til EUs personvernforordning (GDPR) og norsk personopplysningslov.

2. Hvilke opplysninger samler vi inn?

2.1 Skattemelding (PDF/bilde)

Når du laster opp skattemeldingen din, behandler vi følgende opplysninger:

  • Inntektsopplysninger (lønn, pensjon, kapitalinntekt m.m.)
  • Fradragsopplysninger (gjeldsrenter, minstefradrag m.m.)
  • Formues- og gjeldsopplysninger
  • Beregnet skatt og skatteoppgjør

⚠️ Viktig: Skattemeldingen inneholder sensitive personopplysninger. Dokumentet sendes direkte til Anthropics API for analyse og lagres ikke på våre servere etter at analysen er fullført. Vi oppfordrer deg til å lese punkt 5 om underleverandører.

2.2 E-postadresse (betalende kunder)

Dersom du kjøper full tilgang (99 kr) registrerer vi e-postadressen din for å verifisere tilgangen din. Denne lagres i vår database (Supabase) frem til utløpsdatoen (31. desember 2025), eller til du ber om sletting.

2.3 Betalingsinformasjon

Betalinger behandles av Stripe. Vi lagrer ikke kortopplysninger. Vi mottar kun bekreftelse på at betaling er gjennomført, samt e-postadressen du oppga ved betaling.

2.4 Bruksdata (cookies og analyse)

Med ditt samtykke samler vi inn anonymiserte bruksdata via Google Analytics 4 for å forstå hvordan tjenesten brukes. Se punkt 6 om cookies.

2.5 Svar fra fradragsveiviseren

Svarene du gir i fradragsveiviseren (antall barn, pendlingsavstand osv.) brukes kun til å beregne fradraget ditt i den aktuelle sesjonen og lagres ikke.

4. Lagringstid

  • Skattemelding (PDF): Lagres ikke — sendes til Anthropic API og slettes umiddelbart etter svar
  • E-postadresse: Lagres til 31. desember 2025, deretter automatisk slettet
  • Betalingsdata hos Stripe: Lagres i henhold til Stripes retningslinjer (typisk 7 år av regnskapshensyn)
  • Google Analytics: Anonymiserte data lagres i 14 måneder

6. Cookies

Vi bruker følgende typer cookies:

Nødvendige cookies (krever ikke samtykke)

  • LocalStorage (sb_email, sb_pro_2025): Lagrer din pro-tilgangsstatus lokalt i nettleseren din. Lagres kun på din enhet.
  • LocalStorage (sb_kvote_2025): Teller antall gratis analyser du har brukt.

Analyse-cookies (krever samtykke)

  • Google Analytics (_ga, _gid): Anonymisert analyse av bruksmønster. Settes kun ved aktivt samtykke.

Du kan når som helst trekke tilbake samtykket via cookie-banneren på nettsiden.

7. Dine rettigheter

I henhold til GDPR har du følgende rettigheter:

  • Innsyn: Du kan be om innsyn i hvilke opplysninger vi har om deg
  • Retting: Du kan be om at feilaktige opplysninger rettes
  • Sletting: Du kan be om at opplysningene dine slettes ("retten til å bli glemt")
  • Begrensning: Du kan be om at behandlingen begrenses
  • Dataportabilitet: Du kan be om å få utlevert opplysningene dine i maskinlesbart format
  • Innsigelse: Du kan protestere mot behandlingen
  • Trekke tilbake samtykke: Du kan trekke tilbake samtykke til analyse-cookies når som helst

Send forespørsel til: post@skatteberegner.no. Vi svarer innen 30 dager.

9. Sikkerhet

Vi implementerer tekniske og organisatoriske tiltak for å beskytte personopplysningene dine, inkludert:

  • Kryptert overføring (HTTPS/TLS)
  • Begrenset tilgang til personopplysninger
  • Skattemeldingen sendes direkte til Anthropic API og lagres ikke på egne servere
  • E-postadresser lagres kryptert i Supabase

10. Endringer

Vi kan oppdatere denne personvernerklæringen. Vesentlige endringer varsles på nettsiden. Dato for siste oppdatering vises øverst på siden.
